# 📊 Project Overview

This project presents a complete Business Intelligence solution built with Power BI to analyze global sales, profitability, logistics, customer payments, and operational performance.

The dashboard enables stakeholders to monitor key business KPIs, evaluate regional performance, optimize logistics operations, and support strategic business decisions.
